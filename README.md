# Type-Down

A keyboard-first Markdown typing practice app. Improve your typing speed and accuracy by practicing real Markdown syntax.

## Features

- Practice typing Markdown with real-time feedback
- Live Markdown preview as you type
- WPM, accuracy, and time tracking
- Multiple exercise categories (headings, lists, code blocks, etc.)
- Dark mode support
- Progress saved to localStorage

## Getting Started

```bash
npm install
npm run dev
```

## Tech Stack

- React 19 + Vite
- Tailwind CSS v4
- react-markdown
